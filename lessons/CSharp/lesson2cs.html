Условные Операторы

Условный оператор if

В C# и других языках программирования проверка условий является важным аспектом. Мы часто оказываемся в ситуациях, когда необходимо установить, истинны ли условия, чтобы управлять выполнением программы. На эти условия могут влиять ввод пользователя, время, контекст выполнения и прочие факторы.

Конструкция if выполняет блок кода, если заданное условие — истинно, т.е. true.

Синтаксис

if (условие) {
    // Блок кода
}

Условие может возвращать одно из двух значений: true или false.

  • Если условие = true, инструкции внутри тела if (внутри {...}) выполнятся.
  • Если условие = false, C# проигнорирует инструкции внутри тела if.

    • if (number < 3) {
        number += 2;
    }

    В данном примере мы видим фрагмент кода с условием number < 3, т.е. если наше число окажется меньше 3, то мы к нему добавим 2, с помощью оператора +=.

    Оператор a += b является сокращенной формой записи a = a + b

    Условный оператор else

    Иногда требуется выполнить альтернативный код, если условие в операторе if ложно. Для этого используется оператор else. Он позволяет задать блок кода, который выполнится, если предшествующее условие не выполнено.

    if (условие) {
        // Блок кода, если условие истинно
    } else {
        // Блок кода, если условие ложно
    }

    Пример использования if-else:

    if (number < 3) {
        number += 2;
    } else {
        number -= 1;
    }

    В данном примере, если number меньше 3, мы добавим 2, иначе вычтем 1.

    Условные операторы else if

    Для проверки нескольких условий последовательно используется конструкция else if. Это позволяет избежать глубоких вложений и сделать код более читаемым.

    if (условие1) {
        // Блок кода для условия1
    } else if (условие2) {
        // Блок кода для условия2
    } else {
        // Блок кода, если ни одно из условий не выполнено
    }

    Пример использования if-else if:

    if (number < 3) {
        number += 2;
    } else if (number == 3) {
        number = 0;
    } else {
        number -= 1;
    }

    Здесь, если number меньше 3, мы добавляем 2. Если number равен 3, мы устанавливаем его в 0. В противном случае вычитаем 1.

    Условный оператор switch-case

    Когда необходимо проверить одно значение на соответствие нескольким возможным вариантам, удобнее использовать оператор switch. Он улучшает читаемость кода по сравнению с множественными условиями if-else.

    switch (выражение) {
        case значение1:
            // Блок кода для значения1
        break;

        case значение2:
            // Блок кода для значения2
        break;

        default:
            // Блок кода, если ни одно значение не совпадает
    }

    Пример использования switch-case:

    switch (day) {
        case 1:
            // Понедельник
            break;
        case 2:
            // Вторник
            break;
        case 3:
            // Среда
            break;
        case 4:
            // Четверг
            break;
        case 5:
            // Пятница
            break;
        case 6:
            // Суббота
            break;
        case 7:
            // Воскресенье
            break;
        default:
            // Нет такого дня недели
    }

    В данном примере, если day равен 1, выполняется код для понедельника. Если 2 — для вторника, а для остальных значений — блок кода в default.

    case не обозначает какой-либо отдельный блок кода, а только определяет точку входа в него. Когда оператор switch находит подходящий case, программа начнет свое выполнение с этой точки и будет выполнять все последующие блоки кода по порядку. Чтобы этого избежать, необходимо использовать оператор break;, чтобы завершить выполнение оператора switch. switch (month) {
        case 12:
        case 1:
        case 2:
            // Зима
            break;

        case 3:
        case 4:
        case 5:
            // Весна
            break;

        case 6:
        case 7:
        case 8:
            // Лето
            break;

        case 9:
        case 10:
        case 11:
            // Осень
            break;

        default:
            // Нет такого месяца
    }

    В этом примере, если month равен 12, 1 или 2, программа определит, что это зима. Аналогично, для месяцев 3, 4 и 5 будет весна, для 6, 7 и 8 — лето, а для 9, 10 и 11 — осень. Мы смогли это реализовать с помощью подряд идущих case'ов.

    ?: - Тернарный оператор if

    Дополнение

    Тернарный оператор if, или условный оператор, позволяет компактно записывать условные выражения в одну строку. Это удобный способ присвоения значений на основе условия.

    Синтаксис:

    условие ? выражение1 : выражение2;

    Пример использования:

    int number = 5;
    string result = (number > 0) ? "Положительное число" : "Отрицательное число или ноль";

    В этом примере, если number больше 0, переменной result будет присвоено значение "Положительное число". В противном случае ей будет присвоено значение "Отрицательное число или ноль".
