Логическое И и ИЛИ C#

Логические операторы

В C# логические операторы используются для выполнения логических операций над булевыми значениями (true или false). Основными логическими операторами являются:

  • И (&&): возвращает true, если оба условия истинны.
  • ИЛИ (||): возвращает true, если хотя бы одно из условий истинно.

Булевы значения

Булевый тип данных (bool) может принимать только два значения: true или false. Он широко используется в условиях и логических операциях.

Пример использования логических операторов

Рассмотрим пример, в котором мы проверяем, находится ли число в диапазоне от 10 до 20:

int number = 15;
bool isInRange = (number >= 10) && (number <= 20);
if (isInRange) {
    Console.WriteLine("Число в диапазоне от 10 до 20.");
} else {
    Console.WriteLine("Число вне диапазона.");
}

В этом примере мы используем логический оператор И (&&) для проверки, находится ли переменная number в заданном диапазоне. Если оба условия истинны, будет выведено сообщение о том, что число находится в диапазоне.

Рассмотрим ещё один пример с оператором ИЛИ (||):

bool isWeekend = (day == "Суббота") || (day == "Воскресенье");
if (isWeekend) {
    Console.WriteLine("Сегодня выходной!");
} else {
    Console.WriteLine("Сегодня рабочий день.");
}

Здесь мы проверяем, является ли день выходным. Если day равен "Суббота" или "Воскресенье", то программа выведет сообщение о выходном.

Логический оператор НЕ

Логический оператор НЕ (!) инвертирует булевое значение. Если значение истинно, то оно станет ложным, и наоборот.

Синтаксис:

bool isTrue = true;
bool isFalse = !isTrue; // isFalse теперь false

В этом примере переменная isFalse будет содержать значение false, так как оператор ! инвертирует значение isTrue.
Изначально у нас была Прадва - isTrue, потом мы сказали, что Ложь - isFalse равняется НЕ Правде - !isTrue. Примерно так работает НЕ (!) в C#, меняет логическое значение на противоположное.
